# Backlog Wiki Preview for VS Code

VS Code Backlog Wiki Previewプラグインがなくなったため、  
VS Code で Backlog Wiki 記法のリアルタイムプレビューを可能にする拡張機能を簡易的に作成しました。  
macOSでのみ検証。

## インストール方法 (USAGE)

本拡張機能はローカルでビルドしてインストールします。

1.  **ビルドと自動インストール**:
    ターミナルで本ディレクトリに移動し、以下を実行します。
    ```bash
    ./install.sh
    ```
    ※ 内部で `backlog-wiki-preview.vsix` ファイルが生成され、VS Code に自動インストールされます。

2.  **手動インストール (自動で反映されない場合)**:
    `install.sh` 実行後に生成された `.vsix` ファイルを直接 VS Code に読み込ませることもできます。
    *   VS Code を開く
    *   拡張機能タブ（Cmd+Shift+X）を開く
    *   右上の `...` (More Actions) をクリック
    *   `Install from VSIX...` を選択
    *   本ディレクトリの `backlog-wiki-preview.vsix` を選択

3.  **反映**:
    インストール後、VS Code のコマンドパレット（Cmd+Shift+P）から `Developer: Reload Window` を実行するか、VS Code を再起動してください。

## 利用方法

1.  **ファイルを作成する**: 拡張子が `.backlog` または `.bl` のファイルを作成します（例: `test.backlog`）。
2.  **プレビューを開く**:
    *   エディタ右上の **「プレビューアイコン（右側にパネルが開くようなアイコン）」** をクリック。
    *   または、コマンドパレット（`Cmd+Shift+P`）から `Open Backlog Wiki Preview` を選択。
3.  **編集する**: 左側のエディタで Wiki 記法を記述すると、右側のプレビューにリアルタイムで反映されます。
    *   **注意**: プレビューウィンドウは表示専用です。文字の入力は左側のエディタで行ってください。

## 対応している記法

*   **見出し**: `*`, `**`, `***`, `****`, `*****`
*   **リスト**: `-`, `--`, `---`
*   **強調**: `''太字''`, `'''斜体'''`
*   **打ち消し線**: `%%打ち消し%%`
*   **リンク**: `[[タイトル:URL]]` または `[[URL]]`
*   **コードブロック**: `{code} ... {/code}`

## インストール方法

1.  本プロジェクトのディレクトリで `./install.sh` を実行します。
2.  VS Code を再起動（または `Developer: Reload Window`）します。

## うまくいかない場合

*   **プレビューが真っ白**: `.backlog` ファイルを一度クリックしてアクティブ（選択状態）にしてからプレビューを開いてください。
*   **アイコンが出ない**: ファイルの右下にある言語モードが `Backlog Wiki` になっているか確認してください。
